My background / interests

I went into the military immediately following high school (had the grades but couldn't afford college). Along the way, I've gotten quite a bit of training and experience doing software development projects of various types. Early on, as a kid who had only had very basic computer experience (just enough to play games), I definitely struggled. However, as my skill level increased and the resources available expanded dramatically, I've gone from barely getting a programming project off the ground to being able to choose from a variety of APIs and languages to getting involved in things I just dreamt about in the early days.

Computer Vision and Machine Learning

One of my interests developed as I watched the DARPA Grand Challenge races. I saw that one of the teams had attempted a dual camera solution (its vehicle went off the road early in the race, but there's no reason why in the long term we can't use vision). As I was programming in Java at the time, I tried to make programs with that, and as you can imagine, they weren't anywhere close to fast enough with the hardware and the Java VM / way I was programming at the time. Today, after many years of reading blogs and watching videos on online courses from around the world, this interest has pulled me into machine learning as well. These days, I'm writing my own neural net API in Python for the practice, and making automated webscraping scripts to gather the datasets to train the neural net models (my current project is an image classifier using a webscraper that downloads all images related to a keyword). Here are some of my favorite resources in this area:

  • Adrian Rosebrock's PyImageSearch.com PyImageSearch.com (I helped fund his latest Kickstarter to write a book on machine learning with Python)
  • Harrison Kinsley's site: PythonProgramming.net (Also - see his Youtube channel)
  • Professor Andrew Ng's Coursera Machine Learning course (free)
  • Matt Mazur's blog post Step-By-Step Backpropagation: A very verbose description (which was really needed) on how to do gradient descent for a neural net
  • Jevois smart camera: a small, super fast, camera that can run machine learning on the livestream
Here is my project code so far: My machine learning github repo

Helping other coders

Being a programmer can be very frustrating to newcomers. It is extremely difficult to get used to all the troubleshooting and following errors to the source of the problem. The path of the beginner coder's learning curve can be a straight uphill climb-by-your-fingernails type of death march. It doesn't have to be this way. Having a good mentor can mean a world of difference. With this in mind, I have engaged with:

  • CoderVets (helping mentor 40 vets in creating their first mobile & webapp)
  • UMSL (helping with online classes)
  • as a student at LaunchCode with a mind to become a mentor later
